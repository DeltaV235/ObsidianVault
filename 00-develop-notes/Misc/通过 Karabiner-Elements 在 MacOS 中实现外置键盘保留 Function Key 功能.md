title: 通过 Karabiner-Elements 在 MacOS 中实现外置键盘保留 Function Key 功能
created: 2026-03-05
tags:

    - Karabiner-Elements

---

对于由 Karabiner-Elements 发送的 F1-F12 键事件，macOS 将再次将它们转换为媒体键（例如，发送 F10 将触发静音）。如果您想使用复杂修改将 F1-F12 作为原样发送，您需要通过 Karabiner-Elements 发送 fn+F1-F12。
