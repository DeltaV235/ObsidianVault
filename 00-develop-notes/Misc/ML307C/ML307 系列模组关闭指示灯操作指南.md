# ML307 系列模组关闭指示灯操作指南

## 核心信息

**核心指令**: `AT+MLED`

**功能描述**: 设置指示灯状态，配置写入 Flash，**重启后关闭配置仍然生效**。

---

## 1. ML307 Tiny Dongle 专属操作（重点）

针对 **ML307 Tiny Dongle** 这款具体硬件，PCB 板上可见的闪烁指示灯为"网络状态灯"。

### 快速操作

若要完全关闭板载 LED，仅需执行以下一条指令：

```text
AT+MLED=0,0
```

### 效果说明

- **立即生效**: 执行后立即关闭 PCB 板上的 LED 指示灯
- **持久保存**: 配置写入 Flash，重启后依然保持关闭状态
- **简化操作**: 对于 Tiny Dongle 产品，无需配置模组运行灯（即不需要执行 `AT+MLED=1,0`）

### 操作步骤

1. **连接设备**: 通过串口工具（如 PuTTY、minicom、screen）连接 ML307 Tiny Dongle
2. **发送指令**: `AT+MLED=0,0`
3. **预期响应**: `OK`
4. **验证效果**: 观察 PCB 上的 LED 是否停止闪烁/熄灭

## 2. 通用指令参数详解（参考）

如果是在其他开发板或标准模组上使用，涉及的完整参数如下：

### 指令语法

```text
AT+MLED=<func>,<mode>
```

### 参数说明

#### `<func>` - 灯选参数

| 值 | 说明 | 备注 |
|:---|:-----|:-----|
| `0` | 网络状态灯 (NET_LIGHT) | **Tiny Dongle 主要控制这个** |
| `1` | 模组运行灯 (STATUS_LIGHT) | 通常用于开发板 |

#### `<mode>` - 动作参数

| 值 | 说明 | 效果 |
|:---|:-----|:-----|
| `0` | **关闭** (Disable) | 立即熄灭指示灯 |
| `1` | 开启 (Enable) | 恢复指示灯默认行为 |

### 完整操作示例

```text
# 关闭网络状态灯
AT+MLED=0,0
OK

# 关闭模组运行灯（如果硬件支持）
AT+MLED=1,0
OK

# 查询当前灯状态（部分固件支持）
AT+MLED?
+MLED: 0,0
+MLED: 1,0
OK

# 恢复网络状态灯
AT+MLED=0,1
OK
```

## 3. 常见问题排查（Troubleshooting）

### Q: 执行 `AT+MLED=0,0` 后 LED 依然闪烁？

**A**: 
- 检查指令是否正确返回 `OK`
- 确认发送的是正确的 AT 指令（注意大小写）
- 部分硬件可能有多个 LED，确认控制的是正确的灯
- 尝试重启模组后再次观察

### Q: 重启后 LED 又亮了？

**A**: 
- 检查固件版本，部分旧版本固件可能不支持持久化保存
- 重新执行 `AT+MLED=0,0` 并确认返回 `OK`
- 如持续失败，联系厂商确认固件支持情况

### Q: 提示 `ERROR` 或不响应？

**A**: 
- 确认模组已正常启动（通常需要等待 5-10 秒）
- 检查串口波特率设置（默认通常为 115200）
- 尝试先发送 `AT` 测试通信是否正常
- 确认固件版本是否支持 `AT+MLED` 指令

### Q: 如何查看当前灯状态？

**A**: 
```text
AT+MLED?
```
部分固件支持查询功能，会返回当前两个灯的状态。

## 4. 技术背景说明

### 为什么要关闭指示灯？

1. **降低功耗**: LED 闪烁会消耗额外电量，关闭后可延长电池续航
2. **减少光污染**: 在暗环境中使用时，频繁闪烁的 LED 会造成干扰
3. **隐蔽性需求**: 某些应用场景需要设备保持低调运行
4. **专业应用**: 工业环境或批量部署时，统一关闭 LED 可提升用户体验

### Flash 写入机制

`AT+MLED` 指令会将配置参数写入模组的 Flash 存储区域，而非仅修改内存中的临时变量。这确保了：
- **断电保持**: 即使完全断电后重新上电，配置依然有效
- **固件升级保护**: 大部分情况下，固件升级不会清除用户配置区
- **工厂场景**: 适合在生产环节统一配置，无需终端用户干预

## 5. 相关硬件信息

### ML307 Tiny Dongle 产品特点

- **外形**: USB Dongle 形态，即插即用
- **网络**: 支持 LTE Cat.1 网络
- **接口**: USB 2.0，虚拟串口通信
- **指示灯**: PCB 板载单个 LED（网络状态灯）
- **应用**: 物联网网关、工业采集、车载终端等

### 相关产品系列

ML307 系列模组包括多种封装形态：
- **ML307-DCLN**: LCC 封装，开发板常用
- **ML307A**: 增强版，支持更多频段
- **ML307 Tiny Dongle**: USB Dongle 形态（本文档重点）
